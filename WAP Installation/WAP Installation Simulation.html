WAP Installation Simulation

SIMULATION

You are tasked with installing a new wireless access point. This includes running cable, creating patch cords as needed, and ensuring proper connectivity.

INSTRUCTIONS

Part 1: Cable Assembly
For each type of cable shown, drag and drop the appropriate connector type and tool required to prepare and terminate the cable properly.

Part 2: Network Connection
The wireless access point was recently relocated and is no longer connected to the network. Establish a connection between the access point, patch panel, and network switch:

Drag and drop the connectors to the appropriate locations. Once all matching pair is selected (e.g., patch panel to switch), the link will be displayed.

All placeholders must be filled. Cable ends may be reused multiple times.
